# amiibo Checklist
This is just a small fan project to help keep track of my amiibo collection.

This project is not affiliated with, endorsed by, or sponsored by Nintendo. All trademarks and copyrights belong to their respective owners.
It is purely for personal use.

## Features
- Keep track of all your amiibo in an up-to-date checklist.
- Mark your amiibo as "out of box" or "sealed in box" with separate counters for each.
- User-friendly interface for quick updates and tracking.

## How it works
1. Download this repository. (Code -> Download ZIP)
2. Extract the ZIP file.
3. Open `index.html` in your preferred browser.
4. Click on any amiibo to cycle through: Not Owned → Unboxed → Sealed
5. Your progress is saved automatically in your browser.
6. **Tip:** Bookmark the page or create a desktop shortcut for easy access.
